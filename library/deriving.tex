For debuging

\begin{bluetext}
cd `camlp4 -where`
ln -s `ocamlfind query deriving-ocsigen`/pa_deriving.cma 
\end{bluetext}

So you could type \verb|camlp4o -parser pa_deriving.cma test.ml|

\begin{bluetext}

\end{bluetext}
